#+TITLE: Snake game
Educational project in Java and Javax - recreating the snake game with full 360 degree rotation.



** Screenshot
![alt tag](/ScreenshotGame.jpg">



** From command line

   Build it with

   #+BEGIN_SRC sh
     mvn javafx:compile
   #+END_SRC

   Run in with

   #+BEGIN_SRC sh
     mvn javafx:run
   #+END_SRC
